jQuery
- 오픈소스 자바스크립트 라이브러리
    자바스크립트 = HTML + CSS로 만들어진 웹문서를 동적으로 제어하기 위한 언어
- CSS에서 사용되는 Selector(선택자)개념으로 DOM 멤버에 접근
- 웹에서 자주 사용되는 기능을 간편화하여 구현
- 메소드체인 방식에 따른 효율적인 코딩 가능
- CDN(Content Delivery Network)

JSON(Javascript Object Notation)
- 데이터를 저장하거나 전송할 때 많이 사용되는 경량의 DATA 교환 형식을 말한다.
- Javascript에서 객체를 만들때 사용하는 표현식을 사용한다.
- JSON 표현식은 사람,기계 모두 이해하기 쉬우며 용량이 작아서
    최근에는 JSON이 XML을 대체해서 데이터 전송 등에 많이 사용된다.
- JSON은 데이터 포맷
    통신방법 X
    프로그래밍 문법 X
    단순히 데이터를 표시하는 방법 O


jQuery Ajax

Ajax(Asynchronos Javascript and XML)
- 비동기방식의 자바스크립트와 XML
- HTML의 form태그가 아니라 자바스크립트를 통해 비동기 방식으로
    서버에 데이터를 요청한다.
- 서버에서 로딩된 데이터를 출력하기 위해 '새로고침'이 필요없다.
- 필요한 부분만 로딩되므로 속도가 빠르다.

Asynchronos
- 비동기적
- 클라이언트에서 서버에 요청을 보낸 후 응답이 올때까지 대기하지 않고
    프로그램의 흐름은 이어지고, 요청에 대한 겨로가는 콜백 함수로 받게된다.

Javascript
- 자바스크립트 = HTML + CSS로 만들어진 웹문서를 동적으로 제어하기 위한 언어

XML
- Extensible Markup Language 
    : 확장 가능한 마크업 언어
- 사용자의 임의로 태그를 정의할 수 있어 주로 데이터를 표현 및 전달할 때 사용된다.
